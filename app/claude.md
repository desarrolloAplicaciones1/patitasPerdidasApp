# PatitasPerdidas — instrucciones para Claude Code

## Antes de correr la app
1. Verificar que el emulador está iniciado (Pixel 6, Android 17)
2. Build → Clean Project
3. Build → Rebuild Project
4. Run → app

## application ID
com.uade.huellitas  ← debe ser único, no pisarse con otros proyectos

## Stack
Kotlin, Jetpack Compose, Android Studio
